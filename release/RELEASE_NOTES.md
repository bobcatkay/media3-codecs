# Media3 FFmpeg decoder 1.10.1-ffmpeg6.0-r1

Unofficial Media3 FFmpeg audio decoder AAR for Android.

## Contents

- Media3 1.10.1 at commit `5fb306449733dd71595700c1227ad6087578c559`
- FFmpeg `release/6.0` at commit `bda1c6f1f9041910ec5b2c279a08861203a1c95c`
- Enabled decoders: AC-3, E-AC-3, TrueHD, DTS/DTS-HD
- ABIs: `armeabi-v7a`, `arm64-v8a`, `x86`, `x86_64`
- Minimum Android SDK: 23

The AAR is distributed under `Apache-2.0 AND LGPL-2.1-or-later`. The matching
Media3 decoder source, exact FFmpeg source, license texts, build metadata, and
SHA-256 checksums are included with this release.

Suggested Git tag: `v1.10.1-ffmpeg6.0-r1`.
